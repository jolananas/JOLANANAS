# Dossier Fonts — JOLANANAS

Ce dossier contient les fichiers de polices personnalisées utilisées sur le site.

## Police "Weather Sunday - Personal Use"

**Format actuel** : `weather-sunday-personal-use.otf` (OpenType)

### Format disponible
- ✅ `weather-sunday-personal-use.otf` (actuellement utilisé)

### Formats recommandés pour optimisation (optionnel)
Pour améliorer les performances, vous pouvez convertir le fichier `.otf` en formats optimisés :
- `weather-sunday-personal-use.woff2` (recommandé - meilleure compression)
- `weather-sunday-personal-use.woff` (alternative)

### Note
Le CSS référence actuellement uniquement le format `.otf` disponible. Si vous ajoutez des formats optimisés (woff2, woff), mettez à jour `globals.css` pour les inclure en priorité.
